# 🎵 m2mm (MIDI to Mabinogi Mobile MML)

MIDI 파일을 마비노기 모바일 MML 형식으로 변환하는 프로그램입니다.

## 🚀 사용법

1. 프로그램 실행
2. MIDI 파일(.mid)을 드래그하거나 선택
3. 변환 옵션 선택 (일반/화음/악기별)
4. 변환된 MML 코드 복사
5. 게임에 붙여넣기

### 변환 모드

- **일반 변환**: 피치(음높이) 기준으로 파트 분리
- **화음 변환**: 음역대별(멜로디/화음/베이스)로 파트 분리
- **악기별 변환**: MIDI 악기별로 파트 분리

---

## 💡 주요 기능

- **무제한 Voice 생성**: 복잡한 화음도 모두 보존
- **템포 변경 지원**: 곡 중간에 템포가 바뀌어도 정확하게 변환
- **스마트 우선순위**: 음표 수 기반으로 중요한 파트 자동 강조
- **글자수 제한 경고**: 변환이 잘린 경우 명확한 안내

## ❓ FAQ

**Q: MP3는 안되나요?**  
A: MIDI만 지원합니다. MP3는 음표 데이터가 아닌 소리 파형이라 변환이 불가능합니다.

**Q: Voice가 너무 많이 생성돼요**  
A: 복잡한 화음(동시음)이 많으면 10개 이상 생성될 수 있습니다. 초록색으로 표시된 중요한 파트를 우선 선택하세요.

**Q: 변환이 이상해요**  
A: 원본 MIDI 파일 품질을 확인하거나, 원본에 이상이 없음에도 출력이 이상하다면 따로 메일로 원본을 첨부하여 연락주세요.
